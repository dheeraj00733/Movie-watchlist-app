# Movie-watchlist-app
A responsive movie search and watchlist app using OMDB API
# 🎬 Movie Watchlist (Beginner Project)

## 📖 Introduction

This is a basic web project where users can search for movies and add them to a watchlist.

The project is made using simple HTML, CSS, and JavaScript. It uses the OMDb API to get movie data.


---

## 🎯 Objective

* Learn how to use APIs
* Practice JavaScript basics
* Understand how to update UI dynamically
* Build a simple real-world project

---

## ✨ Features

### 🔍 Search Movies

* User can search movies by name
* Data is fetched from OMDb API
* Movies are displayed on the screen

### ⭐ Add to Watchlist

* Each movie has an "Add" button
* Clicking it adds the movie to watchlist

### ❌ Remove from Watchlist

* User can remove movies from watchlist

### 📭 Empty Messages

* Shows message when:

  * No movie is found
  * Watchlist is empty
 
## 🔮 Future Features
- Filter movies by year
- Sort movies alphabetically
- Improve UI design
- Add local storage for saving watchlist

### 🎲 Random Movie (Optional)

* Picks a random movie from watchlist

## 🌐 API Used

OMDb API is used to fetch movie details.

OMDb API Link: https://www.omdbapi.com/

TMDB API: https://developer.themoviedb.org/docs/getting-started

Watchmode API: https://api.watchmode.com/
